# Hi, I'm Janne 👋

I'm a Finnish developer with a slightly unusual path into software. Before I wrote
code professionally, I spent a decade as a bike mechanic and in retail, worked on
a cargo ship, drove trucks, and spent time on construction sites. That background
gives me a different perspective on building things — I care about what actually
works in practice, not just what looks good on paper.

I've been writing code professionally for over a decade now and still genuinely
enjoy it. My day-to-day is full-stack: Node.js, TypeScript, PostgreSQL on the
backend, React on the web, and native Kotlin and Swift on mobile. I've also worn
the CTO hat, which taught me that good software is as much about people and
process as it is about code.

One of my favourite things I've built is SnapCab — a taxi dispatch platform I've
been the sole developer on for four years. It's live, used by 150+ drivers across
Finland, and has its own routing engine. The kind of project that keeps you humble.

Outside of work I run, hit the gym, and go on walks with my Shiba Inu, Fuji.
